# 💕 Love Proposal Website

A beautiful multi-screen romantic proposal website with floating petals, a love letter, reasons carousel, and animated celebration.

---

## 📁 Files

```
index.html   ← The entire website (single file, no dependencies needed)
```

Everything — HTML, CSS, and JavaScript — is in one file. No external assets needed (fonts load from Google Fonts via CDN).

---

## 🚀 How to Publish

### Option 1: GitHub Pages (Free & Easy)
1. Create a free account at [github.com](https://github.com)
2. Create a new repository (e.g. `my-proposal`)
3. Upload `index.html`
4. Go to **Settings → Pages → Source → main branch**
5. Your site will be live at `https://yourusername.github.io/my-proposal`

### Option 2: Netlify Drop (Fastest — no account needed)
1. Go to [netlify.com/drop](https://app.netlify.com/drop)
2. Drag and drop your `index.html` file
3. Done! You'll get a live URL instantly.

### Option 3: Vercel
1. Go to [vercel.com](https://vercel.com) and sign up
2. Click **Add New → Project → Upload**
3. Upload `index.html` and deploy

### Option 4: Traditional Web Host (cPanel)
1. Log into your hosting control panel
2. Open **File Manager → public_html**
3. Upload `index.html`
4. Visit `https://yourdomain.com`
